**#Daily Expense Tracker**

A simple, elegant JavaScript-based expense tracking application with user authentication and localStorage persistence.

**#Project Overview**

This Daily Expense Tracker is a web application that allows users to:

- Login

- Add, view, and delete expenses

- Track total spending in real-time

- Switch between visual themes and export data

**#Features**

- login system 

- Add expenses with description and amount

- Delete individual expenses

- Clear all expenses at once

- Total calculation

- Toggle between Light and Dark themes.

- Download your expense list as a `.csv` file.

**# How to Use**

1.Open login.html in your web browser

2.Use the default credentials

3. Add Expense Enter description and amount, then click "Add Expense"

4. View Expenses: All expenses display with timestamp and amount

5. Delete Expense: Click the trash icon  on any expense

6. Clear All: Click "Clear All" to remove all expenses

7. Logout: Click "Logout" to return to login page
